= Spanish Translation =

Provides a Spanish translation for Elgg.

== Installation ==

- Copy this directory to the mod directory of your Elgg.
- Make sure the owner of 'spanish_translation' is the same user that
  that runs Apache.
- Activate 'spanish_translation' in the Plugins section of
  the 'Administration panel'.
- In 'Basic Settings' of the 'Administration panel' select Spanish
  as Elgg language.
